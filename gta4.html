Grand Theft Auto IV

Overview.

  1. Install RPCS3.
  2. Install GTA IV.
  3. Setup config.
  4. Setup RPCN.
  5. Play!

Install and setup RPCS3.

  1. Download and setup RPCS3.
  2. Download a PS3 system update from playstation.com.
  3. Install the PS3 update using RPCS3 (File > Install Firmware).
  4. Setup your controller (Configuration > Pads).

Install and update GTA IV.

  1. Rip the normal USA version (BLUS-30127) from a disk, or download it from Vimm's Lair.
  2. Add the game to your list of RPCS3 games (File > Add Games). It should be visible in the main window now.
  3. Download the only update for the game from playstation.net.
  4. Install the update using RPCS3 (File > Install Packages).

Setup custom RPCS3 settings for GTA IV (CPU).

  1. Right click GTA IV in your list of games, and select Create Custom Configuration From Default Settings.
  2. Set SPU XFloat Accuracy to Accurate XFloat.

(GPU)

  1. Go to the GPU tab, and ensure the Renderer is Vulkan.
  2. Set Framelimit to Off.
  3. Set ZCULL Accuracy to Precise.
  4. Set Shader Quality to Ultra.
  5. Consider changing Resolution Scale to a lower / higher value depending on your system.
  6. If you changed your Resolution Scale really high, consider turning Anti-Aliasing off.
  7. Set Shader Mode to Legacy.
  8. Turn on Write Color Buffers.
  9. Turn on Multithreaded RSX.
  10. Turn on Asynchronous Texture Streaming.

(Audio)

  1. Go to the Audio tab.
  2. Turn the Audio Buffering Duration to 134 ms. This may require tweaking later if audio sounds bad.
  3. Turn on Enable Time Stretching and set it to 100%.

(Network)

  1. Go to the Network tab.
  2. Set Network Status to Connected.
  3. Set DNS to 8.8.8.8.
  4. Turn on Enable UPNP.
  5. Set PSN Status to RPCN.

(Advanced)

  1. Go to the Advanced tab.
  2. Set Sleep Timers Accuracy to As Host.
  3. Set RSX FIFO Accuracy to Atomic.

(Emulator)

  1. Go to the Emulator tab.
  2. Turn off Show PPU compilation hint.
  3. Turn off Show shader compilation hint.
  4. Consider turning on Enable performance overlay and setting Detail Level to Minimum if you want to see your FPS.
  5. Now that you're done making the custom config, apply and save it.

Play online using RPCN.

  1. Open the RPCN window (Configuration > RPCN).
  2. Click Account and make an account.
  3. After making an account, make sure your account details show up when you click Edit Account.
  4. Make sure the account is valid by clicking Test Account. If it is not valid, go back and edit the account.
  5. Once you start the game, you should now be able to play online using the phone!
