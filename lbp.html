Setup

  1. Download RPCS3.
  2. Download the latest firmware and install it in RPCS3 (File > Install Firmware).
  3. Download all the game you want then add it in RPCS3 (File > Add Games).
  4. Download all the patches for the game you want then install all of them in order (File > Install Packages).
  5. Find the game files in your RPCS3 folder in dev_hdd0/game/(serial)/USRDIR. You'll need to patch EBOOT.BIN to play online.
  6. Change any mentioned settings, and be careful changing others. You can right click the game to give it a custom config if desired.
  7. Setup your controller in RPCS3 (Configuration > Pads).
  8. Setup (and test) an RPCN account (Configuration > RPCN).

Settings

  • In GPU config, enable Write color buffers.
  • In Advanced config, enable Silence all logs.
  • In Network config, set Connected, RPCN, and enable UPNP.
  • If you turned your resolution scale up in GPU config, set the threshold to 1x1.
