Disables the troublemeter.

Arguments

  • Whether to disable the trouble meter or not

Results

None.

Examples

Turn off the punishment system if the player is doing the mission 'Halloween Pranks'

local currentMission = MissionGetCurrentName()

if currentMission == "1.11xP - Halloween Pranks" then -- Checks if the player is currently doing the Halloween Pranks mission with Pete and Gary
	DisablePunishmentSystem(true) -- Disable the Punishment System if doing the Halloween Pranks mission
else
	DisablePunishmentSystem(false) -- Enable the Punishment System if not doing the Halloween Pranks mission
end
