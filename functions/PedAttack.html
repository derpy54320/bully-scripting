Force a specific Ped to attack another specific Ped.

Arguments

  • Ped
  • Target_Ped

    • Results

    None.

    Examples

    Create and make Bif attack Jimmy (the player).

    local x, y, z = PlayerGetPosXYZ()
local ped = PedCreateXYZ(33, x + 1, y, z)
PedAttack(ped, gPlayer)

    Create and make Edgar and Russell fight each other.

    local x, y, z = PlayerGetPosXYZ()
local pedEdgar = PedCreateXYZ(91, x + 1, y, z)
local pedRussell = PedCreateXYZ(176, x - 1, y, z)
PedAttack(pedEdgar, pedRussell)
